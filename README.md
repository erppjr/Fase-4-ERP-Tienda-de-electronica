# ERP Tienda de Electrónica

Sistema de gestión empresarial (ERP) para tienda de electrónica, desarrollado con HTML, CSS y JavaScript vanilla.

## Características

- **Dashboard** - Vista general con métricas y gráficos
- **Inventario** - Gestión de productos y stock
- **Pedidos** - Control de pedidos de clientes
- **Incidencias** - Registro y seguimiento de problemas
- **Proveedores** - Gestión de proveedores
- **Tareas** - Sistema de tareas internas
- **Finanzas** - Control financiero (accesible según rol)

## Roles de Usuario

- **Gerente** - Acceso completo
- **Responsable Almacén** - Gestión de inventario y tareas
- **Empleado Tienda** - Vista de pedidos y tareas

## Tecnologías

- HTML5 / CSS3 / JavaScript ES6
- Chart.js para gráficos
- Phosphor Icons
- Google Fonts (Inter)

## Uso

Abrir `index.html` directamente en el navegador.
